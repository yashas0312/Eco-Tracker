About EcoTracker

EcoTracker is a comprehensive web application designed to help individuals understand and reduce their carbon footprint. Using advanced machine learning models and real-time activity tracking, we provide personalized insights to help you make environmentally conscious decisions.

Built With Modern Technology

  • Frontend: React.js for dynamic user interfaces
  • Backend: Node.js + Express for robust API services
  • Database: MongoDB for scalable data storage
  • ML Service: Python FastAPI + LightGBM for intelligent predictions
